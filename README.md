# Wildes Torres da Silva – Professional Profile

Portfolio profissional publicado via **GitHub Pages**.

## Como publicar no GitHub Pages

1. Acesse **Settings → Pages** no repositório
2. Em **Source**, selecione **Deploy from a branch**
3. Branch: `main` / Folder: `/ (root)`
4. Clique em **Save**
5. Aguarde ~1 min e acesse: `https://<seu-usuario>.github.io/professional-profile`

## Estrutura

```
professional-profile/
├── index.html          ← Página principal do portfólio
├── assets/
│   ├── css/style.css   ← Estilos
│   └── js/main.js      ← Interatividade (scroll, animações)
└── _config.yml         ← Configuração GitHub Pages / Jekyll
```

## Tech Stack do Site

HTML5 · CSS3 (custom) · JavaScript Vanilla · Font Awesome · Google Fonts
